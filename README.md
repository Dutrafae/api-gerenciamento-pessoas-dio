<h2>Desenvolvendo um sistema de gerenciamento de pessoas em API REST com Spring Boot para o Bootcamp da DIGITAL INNOVATION ONe</h2>

O objetivo desse repositório foi desenvolver um pequeno sistema para o gerenciamento de pessoas de umaempresa através de uma API REST, criada com o Spring Boot.

Para executar o projeto no terminal, digite o seguinte comando:

```shell script
mvn spring-boot:run 
```

Após executar o comando acima, basta apenas abrir o seguinte endereço e visualizar a execução do projeto:

```
http://localhost:8080/api/v1/people
```
